## 特徴

* フロント側は完全別のリポジトリとして作成
* クッキーベースのセッション認証を使う
* SPAとAPIはトップレベルドメインが同じで、異なるサブドメインを使う
* 複数のSPAアプリケーションが同じAPIサーバを使う
